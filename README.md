# 📊 Project FORESIGHT

## AI Powered Demand Forecasting & Inventory Intelligence

### Project Description

Project FORESIGHT is a Machine Learning-based application that predicts future medicine demand and helps businesses manage inventory efficiently.

## Features

- Interactive Dashboard
- Demand Forecasting
- Inventory Intelligence
- Stockout Detection
- Overstock Detection
- Download Inventory Report

## Technologies

- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Plotly
